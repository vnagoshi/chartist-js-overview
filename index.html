Chartist.js

Chartist.js is a light-weight charting utility

  • Charts are built as SVG images
  • Chart content is handled via JS, styling is handled by CSS
  • Support for animating chart components
  • Chartist is designed to be fully responsive
  • Appealing default layout with complete freedom to change chart styling
  • Base functionality works on all major browsers

Available at https://gionkunz.github.io/chartist-js/index.html

Chartist.js supports 3 major types of charts

Line Charts

Bar Charts

Pie Charts

Setting up Chartist.js

Download with Bower

bower install chartist --save

CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/chartist.js/latest/chartist.min.css"> <script src="https://cdn.jsdelivr.net/chartist.js/latest/chartist.min.js"></script>

A simple line chart built with Chartist.js

<div class="ct-chart ct-golden-section"></div> <script> var data = {   labels: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'],   series: [     [5, 2, 4, 2, 0]   ] }; new Chartist.Line('.ct-chart', data); </script>

Extended Functionality

There are a number of plugins for Chartist that expand on its charting functionalities

  • Accessability Plugin
  • Tooltip Plugin
  • Zoom Plugin
  • Axis Title Plugin
  • Threshold Plugin

Chartist Plugins with examples are listed on the Chartist.js website
